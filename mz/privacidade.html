Política de Privacidade

Esta página é uma versão simples e serve para a demo. Quando o site estiver em produção, recomenda-se revisão jurídica conforme a operação e os canais utilizados.

Dados que podemos recolher

  • Nome, email e mensagem enviados pelo formulário
  • Informações fornecidas pelo utilizador via WhatsApp

Como usamos os dados

  • Para responder pedidos de orçamento, suporte ou contacto
  • Para organizar entregas e comunicação com o cliente

Partilha de dados

  • Não vendemos nem partilhamos dados com terceiros para marketing
  • Partilha apenas quando necessário para prestação do serviço

Retenção

  • Dados podem ser mantidos pelo tempo necessário para responder e prestar suporte

Contacto

Para dúvidas sobre privacidade, contacte-nos pelo WhatsApp.
